# MauiAppHotel

App pra calcular diarias de hotel. Seleciona o tipo de suite, quantos adultos e criancas, as datas de check-in e check-out, e calcula o valor total da estadia.

**Conceitos novos:** navegacao com PushAsync entre paginas, picker com ItemDisplayBinding, DatePicker com datas minimas e maximas, Stepper pra quantidade, computed properties nos models (Estadia, ValorTotal) e uso de Border com arredondamento.
